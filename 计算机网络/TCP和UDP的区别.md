TCP和UDP的区别

| 类型 | 是否面向连接 | 传输可靠性             | 传输形式   | 传输效率 | 所需资源 | 应用场景           | 首部字节 |
| ---- | ------------ | ---------------------- | ---------- | -------- | -------- | ------------------ | -------- |
| TCP  | 是           | 可靠，保证数据有序     | 字节流     | 低（慢） | 多       | 文件传输、邮件传输 | 20-60    |
| UDP  | 否           | 不可靠，不保证数据有序 | 数据报文段 | 高（快） | 少       | 即时通讯、域名转换 | 8个字节  |
